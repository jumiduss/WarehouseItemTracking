    Filepath Notes
            
        Best Practice uses relative locations
        
        Relative Path Locations
            src="fileName.ending"
                located in same directory folder as the page
            src="parentFolder/fileName.ending"
                located in a folder that's housed in the same directory folder as the page
            src="/parentFolder/fileName.ending"
                located in the directory based on the given root path of the page's directory tree
            src="../fileName.ending"
                located in the parent folder of the page's directory
